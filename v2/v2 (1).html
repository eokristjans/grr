1. Inngangur¶

Fyrirfinnast miss konar gerir af gagnagrindum sem nota m sem lyklasfn. Margar hafa r sna kosti og galla og geta nst vel undir mismunandi kringumstum. Kostirnir geta falist v a vera hravirk, einfld tfrslu, reianleg, ea nota lti geymsluplss, svo dmi su nefnd.

Ein af essum lyklasfnum eru tvleitartr og afbrigi eirra, sem er frgt fyrir a vera hlfgerur jack of all trades, ar sem allar vinslustu aferirnar lyklasafni nota $O(lg(n))$ versta fall ea a jafnai, srstaklega svo framarlega sem tri s jafnvgi. essari skrslu verur skoa bi venjulegt tvleitartr og afbrigi essi, sem ensku kallast Treap, en verur hr nefnt Hrgutr. Hrgutr, eins og nafni gefur til kynna, uppfyllir bi skilyri hrgu, og trs. tfrslan notast vi slembitlugjafa til a thluta hntum forgang vi innsetningu, og hntar me hrri forgang frast ofar tri.

Raaur beintengdur listi er svo anna lyklasafn sem er ekkert srstaklega hravirkt, en me v a tengja saman nokkra raaa beintengda lista m ba til ara gagnagrind sem kallast ensku Skip List, en verur hr kalla Skopplisti. Skopplisti virkar annig a grunninn er einn raaur beintengdur listi me hntum sem innihalda lykil og bendi til hgri. Svo eftir v sem listinn lengist er binn til annar beintengdur listi ar sem hver hntur hefur t.d. 50% lkur a vera essum nja lista fyrir ofan, sama gerist svo vi ann lista egar hann lengist, o.s.frv. Hver hntur essum styttri aukalistum hefur svo bendi niur ann hnt me sama lykil listanum fyrir nean sig. Afleiing essa er s a leit verur mun sneggri, v a arf ekki a tra gegnum allan upprunalega listann til a finna rtta stasetningu, heldur er hgt a taka str skref efri listum og fara svo niur neri lista.

Hrgutr og skopplistar hafa bi ann eiginleika a nta slembigjafa hnnun sinni. Er reianlegt a reia sig slembigjafa hnnun lyklasafna? Hefur str gagnagrindanna hrif reianleika eirra?

1.1 Verkefnalýsing¶

essu verkefni vera tfrar og bornar saman rjr gagnagrindur til a tfra lyklasfn.

Gagnagrindurnar eru:

  1. Tvleitartr (Binary Search Tree)
  2. Hrgutr (Treap)
  3. Skopplisti (Skip List)

Gagnagrindurnar vera tfrar (e. implemented) Python me agerunum:

  • Innsetning (insert())
  • Uppfletting/Leit (contains())

Smaar vera gagnagrindur og ggn.

Aferirnar vera svo tmamldar eftirfarandi mta.

Innsetningin $n_0=500$, $n_1=2500$ og $n_2=5000$ lyklum ar sem gildi lyklana verur $1, 3, 5, ..., 2n_i-1$ (s.s. oddatlur). Til ginda vera lyklasfnin gjarnan kllu lti, mealstrt og strt hr eftir, a gti veitt gott innsi a skoa bi strri og minni gagnagrindur, og $n_2=5000$ s alls ekkert srlega strt.

Run lyklanna vi innsetingu verur tvenns konar:

  • Lyklarnir eru slembir.
  • Lyklarnir eru hkkandi r, burts fr $\frac{n_i}{10}$ handahfskenndum vxlunum (hr eftir: '~hkkandi r')

Fyrir hvora ger af innsetningu vera svo framkvmdar rjr mismunandi tmamlingar leit a $m_0=100.000$, $m_1=200.000$ og svo $m_2=400.000$ lyklum. llum tilfellum vera allir lyklar jafnlklegir til a vera fyrir leitinni, en mist verur

  • Einungis leiti a lyklum sem eru gagnagrindinni (s.s. oddatlum bilinu $[1, 2n_i-1]$), ea
  • Helmingur leitana vera a gildum sem eru ekki gagnagrindinni (s.s. allar heiltlur bilinu $[0, 2n_i]$).

v m taka saman tmamlingar leit tilvikin:

  • Slembin innsetning, allar leitir bera rangur
  • Slembin innsetning, helmingur leitana rangurslaus
  • Nru innsetning, allar leitir bera rangur
  • Nru innsetning, helmingur leitana rangurslaus

Niurstur vera svo birtar tflum og lnuritum, og vera r bornar saman og tlkaar.

# Tvleitartr
class Tree:

    # Smiur
    def __init__(self, data):
        self.data = data
        self.left = None
        self.right = None

    # Innseting
    def insert(self, data):
        if self.data:
            if data < self.data:
                if self.left is None:
                    self.left = Tree(data)
                else:
                    self.left.insert(data)
            elif data > self.data:
                if self.right is None:
                    self.right = Tree(data)
                else:
                    self.right.insert(data)
        else:
            self.data = data

    # Uppfletting    
    def contains(self, key):
        if key < self.data:
            if self.left is None:
                return False
            return self.left.contains(key)
        elif key > self.data:
            if self.right is None:
                return False
            return self.right.contains(key)
        else:
            return True

2.2 Hrúgutré¶

Hrgutri hefur sr klasa fyrir tri sjlft ar sem haldi er utan um hvaa na er rt. Agerir eru framkvmdar tri, sem geymir nur.

Hrgutri uppfyllir hrguskilyri ar sem nur me hrri forgang (lgra gildi self.priority) eru ofar trnu. tfrslan bur ekki upp a hgt s a velja forgang innsetta gildi, en stainn er framkllu handahfskennd fleytitala bilinu $[0,1)$ sem eru u..b. $2^{53}$ lk gildi, og lkurnar a f smu tvisvar v grarlega litlar. Annars vri t.a.m. hgt a upphafsstilla mengi (Python set) af tlum vi smi trsins og fjarlgja gildi r v mengi og nota sem forgang. etta mengi yrfti a a vera a endanlegri str, sem myndi takmarka str trsins.

Leitin er mjg lk leitinni tvleitartr, fyrir utan a fyrst arf a g hvort hrgutri s tmt, og ef ekki, skal hefja leit eins og tvleitartrnu t fr rtinni. a lengir leitartma rlti, en hefur minni hrif eftir v sem tri verur strra (v a vera fleiri agerir hverri leit og essi eina ager er hverfandi).

Sama svo vi um innsetningu hrgutri, nema a ar arf einnig a laga hrguskilyrin eftir hverja innsetningu. v er haldi fram bkinni a snningarnir taki bara fastan tma, en vi munum sj tmamlingunum t.d. hvort innsetning gildum me lyklum slembinni r s raunverulega jafn snggur hrgutrum og tvleitartri.

class TreapNode():
    # Smiur nu
    def __init__(self, data, parent = None):
        self.data = data
        self.priority = random.random()
        self.parent = parent
        self.left = None
        self.right = None


class Treap():
    # Smiur hrgutrs
    def __init__(self):
        self.root = None

    # Uppfletting
    def contains(self, data):
        if self.root is None:
            return False
        return self._contains(data, self.root)
    
    def _contains(self, data, r):
        if data < r.data:
            if r.left is None:
                return False
            return self._contains(data, r.left)
        elif data > r.data:
            if r.right is None:
                return False
            return self._contains(data, r.right)
        else:
            return True

    # Innsetning
    def insert(self, data):
        if self.root is None:
            self.root = TreapNode(data)
            return
        self._insert(data, self.root)
        
    def _insert(self, data, r):
        if data < r.data:
            if r.left is None:
                r.left = TreapNode(data, r)
                self._balance(r.left)
            else:
                self._insert(data, r.left)
        elif data > r.data:
            if r.right is None:
                r.right = TreapNode(data, r)
                self._balance(r.right)
            else:
                self._insert(data, r.right)
        else: # data == r.data
            return

    # laga hrguskilyri
    def _balance(self, s):
        while s.parent is not None and s.priority < s.parent.priority:
            if s.data < s.parent.data:
                self._rotate_right(s)
            else: # self.data > self.parent.data
                self._rotate_left(s)
        if s.parent is None:
            self.root = s
         
    def _rotate_right(self, s):
        p = s.parent
        g = p.parent
        if s.right is not None:
            s.right.parent = p
        if g is not None:
            if p is g.left:
                g.left = s
            else: # p is g.right
                g.right = s
        p.left = s.right
        s.parent = g
        p.parent = s
        s.right = p
                       
    def _rotate_left(self, s):
        p = s.parent
        g = p.parent
        if s.left is not None:
            s.left.parent = p
        if g is not None:
            if p is g.left:
                g.left = s
            else: # p is g.right
                g.right = s
        p.right = s.left
        s.parent = g
        p.parent = s
        s.left = p

        return v is not None and v.key == x


    def insert(self, x):
        h = self.layers
        L = self.sortedLists[h]
        v = L.head
        rVs = []
        while True: # Could put 'v.right.key == x' here instead of True
            if v.right.key < x:
                v = v.right
            elif v.down is not None:
                rVs.append(v)
                v = v.down
            elif v.right.key == x: # but fewer comparisons if kept here
                return
            else: # v.right.key > x and v.down is None. We are here!
                v.right = ListNode(x, v.right, v.down)
                self.updateAbove(x, rVs)
                break


    def updateAbove(self, x, rVs):
        i, n = 0, len(rVs)
        b = self.flipCoin() # flip coin
        while i < n and b: # add x to above layers
            rv = rVs[n-i-1]
            d = rv.down
            while x != d.key: # make sure x is pointing down to itself
                d = d.right
#            print(x, 'inserted with', d.key, 'below')
            rv.right = ListNode(x, rv.right, d)
            b = b and self.flipCoin()
            i+=1
        if i == n and b: # already added to top layer, add another?
            L = SortedLinkedList()
            L.head.down = self.sortedLists[self.layers].head
            L.tail.down = self.sortedLists[self.layers].tail
            self.layers += 1
            self.sortedLists.append(L)

    """ prentar hvern layer af listum L  SkipList  forminu:
        L[0].key (L[0].down.key) -> L[1] (L[1].down.key)...
    """
    def printSkipList(self):

def SmidaLeitarLyklaLista(m, ns, k, b):
    Ls = []
    for i in range(k):
        n = ns[i]
        a = np.arange(0, 2*n) if b else np.arange(1, 2*n+1, 2)
        Ls.append(a)
        while len(Ls[i]) < m:
            Ls[i] = np.concatenate((Ls[i], a))
        random.shuffle(Ls[i])
    return Ls

    for i in range(n//10):

        X[a], X[b] = X[b], X[a]

def MaelaInnsetningartima(DSs, Xs, k, Timi):

""" Tmamla leit
Fyrir:  k er fjldi gagnagrinda (sem  a leita )
        DSs er k-staka listi af gagnagrindum
        Xs er k-staka listi af listum af lyklum

Eftir:  Timi er listi af listum

def MaelaLeitartima(DSs, Xs, ms, k, Timi):

Efri svarti ferillinn er lnulega falli $f(x) = x/100.000$. Hr sst greinilega a innsetning ~hkkandi gilda tvleitartr tekur u..b. lnulegan tma, eins og vnta mtti, ar sem a er nstum v laginu eins og raaur beintengdur listi. Hinir innsetningatmarnir virast einnig vaxa lnulega, en me lgri stuul.

Neri svarti ferillinn er lografalli $g(x) = ln(x)/800$, sem virist vaxa talsvert hgar en hinir ferlarnir. a eru sm vonbrigi, en mgulega mtti finna logra sem vex hraar me annari grunntlu ea rum stuli. a er mislegt sem getur valdi v t.d. a innsetningartminn er tlvu s ekki raunverulega af strargrunni $O(log(n))$. T.d. thlutai tlvan einungis um 30% af rgjrvaaflinu til keyrslu forritsins, og jafnvel au 30% deilast me rum rum sem ku vera vinnslu. Hvernig sem virar eru rir sem eru keyrslu stundum svfir nokkrar millisekndur (ms) v Windows eins og fleiri strikerfi nota Round Robin CPU Scheduling, ar sem ri er thluta svokallaa time slice/quantum sem er gjarnan 40-100ms senn. a ir a margar essar agerir voru framkvmdar 5 til 120 tmaskmmtum. Eins og ur var nefnt eru a fjldi agera vi innsetningu sem er hlutfalli vi $lg(n)$, en tminn tlvu getur veri aeins ruvsi. hugavert vri a prfa arar tmamlinga aferir sem telja ekki me svefntma ra me, flestar heimildir hafi mlt me timeit.

def TaflaLinuheiti(L,I):
    T = []
    for i in range(len(L)):
        T.append(np.concatenate((np.array([L[i]]),I[i])))
    return T

# Br til lista af gildum me leitartma til a birta  tflu
def LeitartimaToflur(L,T,ms):
    Ts = []
    for i in range(len(T[0])):
        Ts.append(
            TaflaLinuheiti(L, [ms,T[0][i],T[1][i],T[2][i]]))
    return Ts

    """ prentar hvern layer af listum L  SkipList  forminu:
        L[0].key (L[0].down.key) -> L[1] (L[1].down.key)...
    """
    def printSkipList(self):
        h = self.layers
        SL = self.sortedLists
        for i in range(h,-1,-1):
            L = SL[i]
            v = L.head
            l = ''
            d = ''
            while v is not None:
                d = str(v.down.key) if v.down is not None else ''
                l += str(v.key)  + ' (' + d + ') ' + '-> '
                v = v.right
            print(l)
            print(d)
            print()
